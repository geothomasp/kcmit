Salary By Period Tab maintenance:

You may do any of the following to populate the fields

•    Manually enter the salary value you wish to appear in the S2S  budget period by typing it in Salary field inthe corresponding Period row in the table.

•    Click the Recalculate button to have the system multiply the maintained base salary on the Details tab by the value maintained in the parameter document DEFAULT INFLATION RATE FOR SALARY

o  After using the recalculate option , you may still manualy adjust the values.

•    Click the Save Changes button to maintain the values and close the window.
